Adding your own HTML

Want to add custom content? Use HTML

HTML

The beauty of Markdown is that you can use HTML with it. (note: you can't mix Markdown inside of HTML).

To add HTML, just paste it in. That's it!

Embeds

Here's some sample HTML. To embed a YouTube video, just paste in the embed code, wrapped in our HTML wrapper block:

<div class="media-element-full">
    <iframe width="420" height="315" src="https://www.youtube.com/embed/4eGQ5VFt7P4" frameborder="0" allowfullscreen></iframe>
    <div class="caption"><p>Caption goes here</p></div>
</div>

which will give you...

Caption goes here

Note: we automatically resize videos to fill the width of the content area for the following services:

  • YouTube
  • Vimeo
  • Blip.tv
  • Viddler
  • Kickstarter
  • Wistia

Tables

You can add tables using HTML. Here's an example table code:

<table>
<caption>Caption</caption>
<thead>
<tr>
     <th>A</th>
     <th>B</th>
     <th>C</th>
</tr>
</thead>
<tr>
    <td>A.1</td>
    <td>B.1</td>
    <td>C.1</td>
</tr>
<tr>
     <td>A.2</td>
     <td>B.2</td>
     <td>C.2</td>
</tr>
</table>

which will give you:

Caption
A B C
A.1 B.1 C.1
A.2 B.2 C.2

3rd Party Widgets

You can add third party widgets by linking to the source files. For example, to add a Google Chart, link to the required source files:

<script type="text/javascript" src="https://www.google.com/jsapi"></script>
<script type="text/javascript">
  google.load("visualization", "1", {packages:["corechart"]});
  google.setOnLoadCallback(drawChart);
  function drawChart() {
    var data = google.visualization.arrayToDataTable([
      ['Task', 'Hours per Day'],
      ['Work',     11],
      ['Eat',      2],
      ['Commute',  2],
      ['Watch TV', 2],
      ['Sleep',    7]
    ]);

    var options = {
      title: 'My Daily Activities'
    };

    var chart = new google.visualization.PieChart(document.getElementById('chart_div'));
    chart.draw(data, options);
  }
</script>

and the HTML it needs to render the output:

<div id="chart_div" style="width: 900px; height: 500px;"></div>
