Media Elements

Make your content come alive

We use what we call Media Elements to add extra design or interactive elements to an article.

These can be elements like pullquotes that create attention and interest to a page, by calling out some text from the main body and pulling it out to the side. You can also add images that float to the side of the body text, to call out certain images.

Media Elements are also used to add multimedia and interactive features, such as image slideshows and embedded video.

We're constantly adding new Media Elements, and here's what you can use today:

Current Media Elements

  1. Pullquote - a piece of text that is emphasized. You can add the quote text and a credit for the author.
  2. Image - an image that's pulled to the side. Upload your image, enter the Alt text (important for accessibility) and add an optional caption and credit.
  3. Video - a video, from YouTube for example. Enter the URL of the video, and a caption.
  4. Audio (Link) - an audio file that's added with a player, for example from SoundCloud. Enter the URL of the audio file, and a caption.
  5. Audio (Embedded) - an audio file that's uploaded. Upload the file, enter the Track Tile, Artist Name, and a link to the band's website.
  6. Slideshow - a slideshow with up to five images. Just upload your images and add optional captions.

Placing your Media Elements

Media elements are added to your content using Markdown shortcodes . These basically let us know where in your content you would like that element to appear.

Woopie isn't a WYSIWYG editor, because depending on what device you're on, What You See might never be What You Get. Placing a media element is more about where in the flow of your content you want it to appear, rather than its exact location on the page.

Just copy the shortcode - it looks like this -
![media_123] - and paste it into the Content block. If you want to move it, delete it and paste it somewhere else. If you don't place in a shortcode, it will not appear in the content.

To delete a Media Element, first delete the shortcode from the content, and then click Remove this item on the relevant element.
