Add formatting and style to your content

Markdown is a quick and simple way to markup text - that is, to describe what text is - whether it's a heading, or a list, or a quote. Created by John Gruber it uses many conventions you may be familiar with from text and email.

How to use Markdown

Headings

Headings are created by adding a number of # in front of your text. The number of # indicates the level of heading, one being the highest. For example:

# Create Heading 1
## Create Heading 2

This is a Heading 1

This is a Heading 2

Paragraphs

Paragraphs are created by default, and separated by one or two blank lines.

Lists

You can create an unordered list (a bulleted list) by placing an * or a + or a - before each item. For example:

* Item One
    * Create a sub-list by indenting with four spaces
* Item two
* Item three

Numbered lists are easy too - just use numbers.

  1. An item in an ordered list
    1.1. A subitem, indented with 4 spaces
  2. Another item in an ordered list

Blockquotes

Place a > in front of text to turn it into a blockquote

This is an example of a blockquote. A blockquote is a good way to emphasis a piece of text.

Emphasising Text

Make text italic by using single asterisks *italic* and make text bold using double asterisks **bold**

Links

Create a link like Woop.ie by putting the link text in [square brackets] followed by the link address in (round brakets) - like so
